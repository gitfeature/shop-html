# Shop HTML

一个简洁的电商HTML模板项目。

## 项目结构

```
shop-html/
├── index.html      # 首页
├── products.html   # 产品列表页
├── product.html    # 产品详情页
├── cart.html       # 购物车页面
├── checkout.html   # 结账页面
├── login.html      # 登录页面
├── user.html       # 用户中心页面
├── css/            # 样式文件
├── js/             # JavaScript 文件
└── images/         # 图片资源
```

## 功能特点

- 响应式设计，适配多种设备
- 完整购物流程：浏览、加入购物车、结算
- 用户认证系统
- 产品展示与管理

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)

## 运行方式

直接在浏览器中打开 `index.html` 即可预览。
